## ✨ Sobre mim

Sou estudante de tecnologia em Pernambuco, em formação na área de programação, desenvolvimento web e soluções digitais.

Estou construindo minha trajetória com projetos acadêmicos, estudos práticos e organização dos meus aprendizados no GitHub. Meu objetivo é evoluir como desenvolvedora, criando soluções úteis e transformando conhecimento em projetos reais.

- Git & GitHub para versionamento, organização de repositórios e publicação de projetos.
- Power BI e Google Sheets para organização, análise e visualização de dados.
